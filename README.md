## Features 🔥
- Fully generated C# SDK based on [Bland AI API](https://bland.ai) using [AutoSDK](https://github.com/HavenDV/AutoSDK)
- All modern .NET features - nullability, trimming, NativeAOT, etc.
- Targets .NET 10.0

### Usage
```csharp
using BlandAI;

using var client = new BlandAIClient(apiKey);
```
